The Portfolio of Charlie Chen

Projects

  • Project 1 : CSE Simulator (Unreal Engine)

    This project used Unreal Engine 5, Blueprinting, Affordance, Mixamo, and Sketchfab.

  • Project 2 : A2-Go AR Exploration Game

    An AR-based exploration game where players plant trees, earn currency over time, and interact with the environment. Includes a custom “Raining Acorn” powerup that spawns acorns from the sky, adding a dynamic and engaging gameplay element.

  • Project 3 : SkyEase VR (Exposure Therapy)

    A VR exposure therapy application for treating aerophobia (fear of flying). Built in Unreal Engine 5, it features a realistic airplane cabin and progressive exposure levels that help users gradually build tolerance to flight-related anxiety in a safe, controlled environment.
